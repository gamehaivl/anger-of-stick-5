# Tải Anger of Stick 5 Hack (Vô Hạn Tiền, Kim Cương) v1.1.94

Bộ source tĩnh GitHub Pages cho bài viết **Tải Anger of Stick 5 Hack (Vô Hạn Tiền, Kim Cương) v1.1.94** trên GAMEHAIVL.

## Cấu trúc

```text
.
├── index.html
├── anger-of-stick-5/index.html
├── 404.html
├── robots.txt
├── sitemap.xml
├── .nojekyll
└── README.md
```

## Tối ưu đã có

- HTML tĩnh, không phụ thuộc WordPress/PHP/jQuery/Flatsome/wpDiscuz.
- CSS và JS nhúng trực tiếp để hạn chế lỗi load asset trên GitHub Pages.
- Giữ title, meta description, canonical, Open Graph, Twitter Card.
- Có schema Organization, WebSite, WebPage, Article, SoftwareApplication, FAQPage, BreadcrumbList.
- Chỉ dùng 1 H1, heading H2/H3 rõ ràng.
- Ảnh dùng URL tuyệt đối từ GAMEHAIVL và có fallback khi ảnh lỗi.

## Cách deploy

1. Giải nén ZIP.
2. Upload toàn bộ file bên trong thư mục này lên root repository.
3. Vào Settings > Pages.
4. Chọn branch `main` và thư mục `/root`.
5. Đảm bảo `index.html` nằm ngay ngoài cùng repo.
